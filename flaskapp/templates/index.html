Secure DevOps App Deployed Successfully

This project, completed by Team 17, demonstrates the deployment of a secure web application built with either Flask or Node.js. The application is hosted on an EC2 instance running Ubuntu 20.04, with Jenkins managing the build and deployment pipeline from a master server on Amazon Linux and an agent on CentOS 8.

A key focus of this project is security: the app is served over HTTPS using Nginx and Let's Encrypt, and it connects to a MongoDB backend populated with starter data. Before deployment, Jenkins archives the application along with a SHA-256 hash file (hash.txt), and Ansible verifies this hash to ensure artifact integrity.

The result is a production-grade application securely deployed and accessible over HTTPS on port 443, displaying the message: "Secure DevOps app deployed with verified artifact!"
