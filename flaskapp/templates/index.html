Secure DevOps App Deployed Successfully

This project, completed by Team 17, demonstrates the deployment of a secure web application built with either Flask or Node.js. The application is hosted on an EC2 instance as well as the Jenkins master and Jenkins Agent.

A key focus of this project is security: the app is served over HTTPS using Nginx and Let's Encrypt, and it connects to a MongoDB backend populated with starter data.
